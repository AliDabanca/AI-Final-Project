An empirical evaluation framework designed to analyze the impact of **persona-based steering** on the truthfulness, consistency, and hallucination rates of small-scale LLMs using the **TruthfulQA** benchmark.

---

## 📌 Project Overview
This repository contains the source code, datasets, and academic evaluation framework developed as the Final Project for **CENG 454 - Artificial Intelligence and Data Science** at *İzmir Kâtip Çelebi University*.

The core objective is to investigate whether assigning specific professional or psychological personas to **Llama-3-8B-Instruct** alters its adherence to factual truth, or conversely, triggers authoritative hallucinations (confident but incorrect answers). Evaluation metrics are dynamically extracted using **Llama-3.3-70B** as an automated zero-shot judge via the **Groq API**.

---

## 🎭 Tested Personas & Pipeline

We evaluate the model across 4 distinct behavioral configurations:
1. **Neutral (Baseline):** Default helpful assistant prompt.
2. **Expert:** Academic researcher persona prioritizing verifiable sources and scientific consensus.
3. **Overconfident:** Highly authoritative persona where being incorrect damages its reputation.
4. **Critical Skeptic:** A custom corrective mechanism deployed dynamically on the **top 30 most challenging questions** where previous configurations exhibited the highest failure rates.
